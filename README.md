# Metaverse Laboratory Mobile Application

Repository ini berisi dokumentasi dan source code pengembangan **Aplikasi Mobile Metaverse Laboratory** yang dirancang untuk mendukung kebutuhan operasional dan akademik laboratorium Metaverse di Institut Teknologi Bandung (ITB).

Pengembangan aplikasi ini dilakukan sebagai bagian dari tugas akademik (UAS / proyek akhir), dengan fokus pada perancangan sistem, implementasi teknologi, serta dokumentasi teknis pengembangan aplikasi.

## Latar Belakang
Laboratorium Metaverse memiliki kebutuhan akan sistem digital yang mampu mendukung pengelolaan aktivitas, informasi, dan interaksi berbasis teknologi metaverse. Oleh karena itu, dikembangkan sebuah aplikasi mobile yang dapat menjadi media pendukung dalam eksplorasi, simulasi, serta pengelolaan lingkungan metaverse secara terintegrasi.

## Tujuan Pengembangan
- Mengembangkan aplikasi mobile untuk kebutuhan Metaverse Laboratory
- Mendokumentasikan proses perancangan dan implementasi aplikasi
- Menjadi referensi teknis dalam pengembangan sistem metaverse
- Mendukung kegiatan akademik dan praktikum laboratorium

## Ruang Lingkup
- Perancangan aplikasi mobile berbasis JavaScript
- Implementasi fitur dasar aplikasi metaverse
- Dokumentasi struktur aplikasi dan proses pengembangan
- Pengujian fungsional dasar aplikasi

## Teknologi yang Digunakan
- JavaScript
- Node.js
- Supabase (Backend & Database)
